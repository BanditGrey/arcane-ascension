## 📦 Sistemas Implementados

### Core
- Motor de jogo (Canvas + TypeScript)

### Progressão
- Sistema de Proficiências (nível automático)
- Sistema de Especializações (15+)
- Árvore de Habilidades (Skill Tree)
- Sistema de Equipamentos + Stats

### Combate
- 8+ magias com VFX por elemento
- Hotbar funcional (teclas 1-4)
- Sistema de Partículas avançado
- Bosses com vida e fases

### Economia & Loot
- Sistema de Loot
- Sistema de Sets de equipamentos
- Crafting (base)

### Persistência
- Save/Load (localStorage)
- Teclas **K** = Salvar / **L** = Carregar

### Visual
- Animações do mago (idle/walk/cast)
- Aura + partículas constantes
- Efeitos visuais por elemento

## 🎮 Controles Finais

| Tecla       | Ação                          |
|-------------|-------------------------------|
| WASD        | Movimento                     |
| Clique      | Lançar magia atual            |
| 1, 2, 3, 4  | Trocar magia (Hotbar)         |
| K           | Salvar jogo                   |
| L           | Carregar jogo                 |
