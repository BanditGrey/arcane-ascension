# 🔥 Arcane Ascension

**RPG de navegador completo** focado em um único Mago.

Jogo profundo inspirado em Path of Exile, Diablo e Last Epoch — mas 100% centrado na evolução mágica.

### ✨ Principais Funcionalidades

- Evolução visual do mago (4 estágios de aparência)
- Sistema de cajados dinâmicos com efeitos visuais
- Bônus permanentes por estágio de evolução
- 8 tipos de cajados diferentes
- Sistema de proficiências e especializações
- Torre Infinita

---

## ✅ O que já foi implementado

- Sistema de movimento WASD fluido
- Sistema de projéteis + colisão com inimigos
- 5 tipos de magias (Fogo, Gelo, Raio, Arcano, Trevas)
- Sistema de Proficiências (ganha XP automaticamente)
- Sistema de Especializações (escolha de classe)
- Inventário funcional
- Spawn contínuo de inimigos
- HUD dinâmico
- Torre Infinita (andares infinitos)

## 🚀 Como rodar o jogo

```bash
npm install
npm run dev
```

Abra o navegador em `http://localhost:3000`

## 🎮 Controles

- **WASD** → Movimento
- **Clique** → Lançar Bola de Fogo

## 📦 Estrutura

```
src/
├── game/           # Núcleo do jogo
├── data/           # Spells, Items
├── systems/        # Proficiência, Inventário, Especializações
└── ui/             # Gerenciador de HUD
```
